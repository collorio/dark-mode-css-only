Modo escuro utilizando somente HTML e CSS




É possível criar o efeito de "dark mode" utilizando somente o CSS e o HTML com o atributo checked e tags input radio, combinados às variáveis do CSS.


No arquivo CSS desse mini-projeto temos quatro padrões de cores disponíveis sem utilizar nada de Javascript, tudo no CSS puro.




O elemento has() do CSS funciona como uma condicional (if), checando qual dos seletores está marcado e aplicando, por meio das variáveis declaradas, as alterações de cores representativas daquele seletor.




A declaração de variáveis no CSS se dá utilizando, direto no arquivo CSS: 

 elemento {
            --cor-bg-principal: brown;
          }


Para utilizar a varíavel a sintaxe é: 

  elemento {
            background-color: var(--cor-bg-principal);
          }




Para uma explicação mais detalhada é possível consultar o MDN Web.




Deve ser observado que o has() ainda não é aceito por todos os navegadores, sendo necessário fazer um failsafe com javascript para garantir o funcionamento desta funcionalidade. É possível conferir pelo site Can i use quais navegadores aceitam, ou não, essa funcionalidade.
